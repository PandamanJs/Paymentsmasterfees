# School Logos

This folder contains logo images for the schools in the Master-Fees application.

## Required Logo Files

Place your school logo images in this directory with the following names:

- `chimilute-logo.png` - Chimilute Trust Academy logo
- `julani-logo.png` - Julani School logo
- `crested-crane-logo.png` - Crested Crane Academy logo
- `maarif-logo.png` - International Maarif School logo
- `twalumbu-logo.png` - Twalumbu Educational Center logo (optional)

## Image Specifications

- **Format**: PNG or JPG
- **Recommended Size**: 120x120 pixels (or larger, will be scaled down)
- **Background**: Transparent PNG preferred
- **Quality**: High resolution for crisp display on all devices

## Usage

These logos will appear in:
- School search dropdown
- School selection pages
- Payment receipts
- Student details pages

If a logo file is not found, the school name will display without a logo icon.
